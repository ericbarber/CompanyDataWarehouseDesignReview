# Glossary

| Term | Definition |
|---|---|
| CDC | Change Data Capture; propagation of inserts, updates, and deletes from a source system |
| CUI | Controlled Unclassified Information |
| Data contract | Versioned agreement describing a dataset's structure, semantics, quality, and operating expectations |
| Data product | An owned and supported dataset or service designed for consumer use |
| FCI | Federal Contract Information |
| Idempotent | Safe to repeat without producing unintended additional effects |
| Micro-batch | Processing small groups of records at short, repeated intervals |
| ODS | Operational Data Store containing integrated operational data, commonly representing current state |
| RLS | Row-Level Security |
| Trust boundary | A boundary across which identity, authorization, or data-protection assumptions change |
| Watermark | A value used to track processing progress through an ordered data source |
