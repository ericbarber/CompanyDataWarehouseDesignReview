# Notebook Catalog

The initial repository scan found **581 Python notebook files**. The catalog will be generated and then reviewed rather than maintained manually.

## Required notebook definition

- Notebook ID, name, and repository path
- Layer and business domain
- Purpose and status
- Calling jobs and notebooks
- Widgets and parameters
- Tables, files, APIs, and secrets read
- Tables, files, APIs, metadata, and permissions written
- Imported notebooks and functions
- Error and exit behavior
- Responsible owner
- Modernization disposition
