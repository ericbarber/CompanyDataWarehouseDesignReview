# Draw.io Diagram Sources

Store editable draw.io files here using the architecture view identifier and a descriptive name:

```text
A00-enterprise-context.drawio
A01-current-logical-data-flow.drawio
A02-physical-deployment.drawio
A03-orchestration.drawio
A04-<domain>-data-flow.drawio
A05-security-boundaries.drawio
A06-<processing-pattern>.drawio
A07-target-architecture.drawio
A08-transition-architecture.drawio
```

Export reviewed diagrams as SVG to `src/assets/diagrams/` and embed the SVG from its corresponding Markdown page. Do not discard the editable `.drawio` source.
